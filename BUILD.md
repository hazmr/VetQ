# VetQ — تشغيل وبناء التطبيق

تطبيق عيادة بيطرية محلي (Electron + React + SQLite). يعمل بدون إنترنت، يدعم **العربية والإنجليزية** (زر تبديل اللغة، الافتراضي عربي)، مع وضع فاتح/داكن. **مخصّص لويندوز فقط.**

---

## بناء ملف التثبيت على ويندوز (.exe)

كل شيء يتم على جهاز **ويندوز**.

### 1. المتطلبات (مرة واحدة)

- **Node.js 20 أو أحدث** — حمّله من <https://nodejs.org> (نسخة LTS).
  أثناء التثبيت فعّل خيار **"Automatically install the necessary tools"** (يثبّت أدوات البناء + Python) لضمان بناء وحدة قاعدة البيانات `better-sqlite3`.

### 2. انسخ مجلد المشروع إلى الجهاز

انسخ كامل مجلد المشروع (بدون مجلدات `node_modules` و `release` و `out` إن وُجدت — سيُعاد إنشاؤها).

### 3. افتح PowerShell أو CMD داخل مجلد المشروع ونفّذ:

```powershell
npm install
npm run pack:win
```

- `npm install` يثبّت الحزم ويبني وحدة قاعدة البيانات تلقائيًا لويندوز.
- `npm run pack:win` يبني التطبيق وينتج ملف التثبيت عبر electron-builder.

### 4. ستجد الناتج في مجلد `release`:

- `VetQ Setup 1.0.0.exe` ← **ملف التثبيت** الذي توزّعه على أجهزة العيادة.
- مجلد `win-unpacked` ← نسخة محمولة تعمل مباشرة بدون تثبيت (`VetQ.exe`).

> ملف التثبيت يتيح اختيار مكان التثبيت وينشئ اختصارًا على سطح المكتب.

## التطوير

```powershell
npm install
npm run dev      # يشغّل التطبيق في وضع التطوير مع إعادة التحميل
npm run build    # يبني فقط دون تغليف
```

---

## اللغة والمظهر

- زر **🌐 اللغة** يبدّل بين العربية والإنجليزية (الافتراضي عربي). الاتجاه ينقلب تلقائيًا RTL/LTR.
- زر **الوضع الليلي/النهاري** يبدّل المظهر.
- كلا الإعدادين يُحفظان ويظهران أيضًا في صفحة **الإعدادات → المظهر واللغة**.

---

## أين تُحفظ البيانات؟

- قاعدة البيانات: مجلد بيانات المستخدم لويندوز
  `C:\Users\<اسمك>\AppData\Roaming\vetq\vetq.db`
- النسخ الاحتياطية التلقائية: داخل نفس المجلد في `backups\` (أو المجلد الذي تختاره من شاشة "النسخ الاحتياطي").

عند الاستعادة أو النقل لجهاز آخر، انسخ ملف `vetq.db` أو استخدم زر "نسخة احتياطية الآن" ثم "استعادة من ملف".

---

## أيقونة التطبيق (اختياري)

لتغيير الأيقونة الافتراضية: ضع ملف `build/icon.ico` (256×256) قبل تنفيذ `npm run pack:win`.

---

## ملاحظة عن الطابعة

الطباعة تتم عبر تعريف الطابعة في ويندوز (يدعم العربية والإنجليزية تلقائيًا). من شاشة **الإعدادات → الطباعة** اختر الطابعة وفعّل "طباعة تلقائية عند الضغط". يعمل مع طابعات الإيصالات الحرارية 58/80 مم المثبّتة في ويندوز. لغة الإيصال تتبع لغة الواجهة.
